My Work

Flaneur

Flaneur is an interactive web-based application that allows users to log in, search for a city and build a custom, optimized walking route allowing them to see as many or as few attractions available in that city.

Technologies Used:

  • Google Maps & Yelp APIs
  • Mongo/Mongoose Database
  • Passport
  • React-Bootstrap
  • Heroku

GifTastic

GifTastic is a car-themed web application where users can click a button to load gifs related content and create buttons of their own. They can also start and stop the gif by clicking on the image.

Technologies Used:

  • Giphy API
  • Javascript/Jquery
  • Bootstrap
  • Heroku

Music Events Near You

Music Events Near You allows user to log in and search for upcoming music events based on genre and geolocation. They can also see music samples for an artist.

Technologies Used:

  • SeatGeek & Youtube APIs
  • Firebase
  • Bootstrap
  • Heroku

Train Scheduler

Train Scheduler is an web-based application that allows users to see current train times updating in a real time. The user also has the capabilities to add their own train and the page will update dynamically to show the next train.

Technologies Used:

  • Firebase
  • Javascript/Jquery
  • MomentJS
  • Bootstrap
  • Heroku

Friend_Finder

Friend Finder allows users to submit a questionaire where their answers are analyzed using an alogorithm to match them with a new friend.

Technologies Used:

  • NodeJS
  • Express
  • Bootstrap
  • Heroku

Sequelize Burger

An application where the user can dynamically add burgers and the webpage will update with the burger name and a devour button. The burger name is added to the database.

Technologies Used:

  • MySql/Sequelize Database
  • NodeJS
  • Handlebars
  • Heroku
